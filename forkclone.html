Pasos para Realizar un Fork y Clone en GitHub

Paso Descripción
1 Visita el repositorio en GitHub que deseas forkear.
2 Haz clic en el botón "Fork" en la esquina superior derecha de la página. Esto creará una copia del repositorio en tu cuenta de GitHub.
3 Abre la ventana de comandos en el directorio donde deseas clonar el repositorio.
4 Clona tu repositorio forkeado a tu máquina local usando el comando git clone <URL_del_repo_fork>.

Diferencia entre Fork y Clone

Fork: Hacer un fork en GitHub significa crear una copia independiente de un repositorio en tu propia cuenta de GitHub. Puedes trabajar en tu versión del repositorio sin afectar el original.

Clone: Clonar un repositorio significa descargar una copia del repositorio (ya sea el original o un fork) a tu máquina local. Esto te permite trabajar con el código en tu entorno de desarrollo local.
